Using React to make a movie site.

This is a bit more of a look at this site as a case study. This site uses React to pull the images, info and any changes made to the site. Buttons at the top of the page are to sort movies by rating, title, year and release date. Site uses these to show how React changes and updates information on the backend of the site. Rather than make a separate page, it makes the changes on the backend and displays the results. This is faster and more simple to build than makeing a page for every option. This also makes it so that when movies are added, it updates it with out needing to refresh and making new pages for them. Saving time, space, and bandwidth.
